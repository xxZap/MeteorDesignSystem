# Features

## Comprehensive

Composed of a total of 11 crafted components that you can use out of the box.

## SwiftUI friendly

All components are written in SwiftUI.

## List of components

### Actions
- Buttons
  - Button
  - IconButton
  - SideButton
- ListItem

### Foundations
- Animations
- Colors

### Inputs
- TextField

### Layout
- InfoBox
- Progress
- Tooltip

### Loading
- Spinner
